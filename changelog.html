  • added countrycode to station information, country is deprecated but will function for some time
  • added bycountrycodeexact listing of stations
  • added list of countrycodes

  • changed list takes lastchangeuuid instead of seconds parameter
May 9, 2018
  • Language field is multivalue now like tags field.
  • Fixed hidebroken parameter for many lists.
  • Allow get parameters for many lists.
April 12, 2018
  • Online check for edit must turn out ok, for the station to be saved.
March 19, 2018

  • Restricted station name to 400 chars
  • bugfix: initialize HLS on stream check

February 18, 2018

  • Added api to get results of station checks.(Documentation)

February 2, 2018

  • Deactivated undelete, delete and revert for now. some bugs have to be investigated to activate it again.

January 30, 2018

  • Properly encode JSON values with json_encode
  • Added stationuuid and changeuuid to station list entries. This will be used in future for cross server replication of entries. The api will migrate to also support uuids. Old ids will also be supported but on the long run at some point deprecated.
  • Added seconds parameter to stations/changed api. This api call does now also return the current active station in addition to the historical data in a single list. Before it only contained backups without the current state. This helps with keeping server sync simple.

December 14, 2017

  • Sanitize urls (homepage, favicon, url) by replacing spaces with %20.

September 20, 2017

  • save ip of sources of changes

August 19, 2017

  • Clicks are now counted only once per day per ip address.

April 21, 2017

  • Turtle/RDF support for station lists added.
    Example: http://www.radio-browser.info/webservice/ttl/stations/byname/bbc
    Tested against: http://ttl.summerofcode.be/

March 18, 2017

  • M3U, PLS and XSPF playlist support for station lists.
    This means you can paste an url like
    http://www.radio-browser.info/webservice/pls/stations/topclick/5
    into your favourite audio player. (for example VLC)

March 3, 2017

  • Hls stream detection support. Streams have to be decoded on the client.

January 21, 2017

  • Fixed " characters in name handling in some api calls. (e.g.: /webservice/v2/json/url)
